Justificación de las estructuras de datos (tiempos estimados de corrimiento) y de los algoritmos de ordenamiento (la elección y el diseño):
Para este proyecto se hizo uso de arboles AVL, principalmente el metodo de busqueda, eliminar y agregar elementos, que tienen un tiempo estimado
de corrimiento del orden de logn, lo cual simplifica enormemente la busqueda de contenedores donde el objeto quepa.
Por otro lado tambien se hizo uso de listas ligadas para almacenar algunos datos, en los metodos utilizados se espera un tiempo de corrimiento del
orden de n, generalmente son usadas en aplicaciones donde se requiere recorrer todos los elementos, por ejemplo para mostrarlos o hacer busquedas.
En cuanto a los algoritmos de ordenamiento se hizo uso de quicksort por la simplesa de implementacion y debido a que es del orden de nlogn, lo cual
es mas que aceptable para los propositos del proyecto.
